# Supernova Launcher

> **Status:** Work in progress

Supernova Launcher is a feature-rich, customizable Minecraft launcher focused on quality-of-life improvements and seamless instance management.

---

## Features

- **Instance Management:** Easily create, organize, and run custom Minecraft instances.
- **Cross-Instance File Syncing:** Sync options, saves, and settings between instances ([Watch Demo](https://youtu.be/wb5EY2VsMKg)).
- **Mod Deduplication:** Hard-link duplicate mods installed through the launcher to save disk space.
- **Modpack Management:** A unique approach to managing and updating modpacks ([Watch Demo](https://youtu.be/cdRVqd7b2BQ)).
- **Integrated Mod Browsing:** Browse and download mods directly from both Modrinth and CurseForge.
- **Built-in Skin Editor:** Customize your player skin and cape directly within the launcher.
- **Secure Credentials:** Keep account tokens safe using native platform keyrings.
- **Automatic Log Uploads:** Instant log sharing via [mclo.gs](https://mclo.gs) with automatic redaction of sensitive data (e.g., access tokens).
- **Customization:** Fully customizable color schemes.
- **Custom Game Output Window:** Easily view game logs while playing.
- **Pinned Worlds & Servers:** Quickly jump back into your favorite worlds or servers.
- **Recently Played List:** Pick up right where you left off.

## FAQ

### Where can I suggest a feature or report a bug?
Feel free to open an issue on our [GitHub Issues](../../issues) page or join our [Discord Server](https://discord.gg/CngWzd4bKh).

### Why should I use Supernova over other launchers?
Use whatever launcher you like best! Supernova just offers a few convenient QoL features.

### Will Supernova be monetized?
No, Supernova is and will remain free and ad-free.

- I believe that it is wrong for launchers to be monetized without distributing revenue back to mod creators that give the launcher value in the first place. Since I don't have the infrastructure to be able to redistribute revenue to mod creators, this is a big barrier.
- Dealing with monetization takes a lot of (ongoing) work, probably more work than creating the launcher itself.
- I personally dislike advertisements.
